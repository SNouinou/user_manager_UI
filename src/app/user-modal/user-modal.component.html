Welcome to UserManager!

UserManager is a powerful application designed for managing users, their roles, and access rights. This app is created to showcase user management functionalities and provide a hands-on experience with Spring Security and Angular.

Getting Started:

  1. Request a User List: Begin by generating a random list of users through the app. This list will serve as your initial dataset for managing user roles and permissions.
  2. Upload Users: Once you have your list, upload it to the app. This is the sole method for adding users to the system.
  3. Log In and Manage: After uploading, log in to the app. As an admin, you'll have comprehensive control over user access and roles. You can modify permissions, assign roles, and even delete users as needed.

Features:

  • Admin Controls: Admin users have full authority to manage other users, including setting roles and adjusting access rights.
  • Role-Based Access: Define and manage various user roles to control what each user can see and do within the app.

Note: This app serves as a demo for a coding exercise and showcases practical use of Spring Security and Angular. Dive in and explore the robust user management capabilities!
